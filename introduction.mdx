## Visão Geral

A 3x Change é uma API que facilita a conversão de PIX para criptomoedas de forma simples e segura. Nossa solução permite que você integre facilmente operações de conversão de moeda brasileira para criptomoedas em sua aplicação.

## Como Funciona

O fluxo da 3x Change é simples e direto:

1. **Criação de Carteira**: Crie uma carteira cripto passando os parâmetros necessários
2. **Cotação**: Consulte a cotação atual das criptomoedas disponíveis
3. **Pagamento PIX**: Gere um pagamento PIX e, após confirmação, receba automaticamente a criptomoeda
