---
title: "Listar Contas Bancárias"
openapi: 'GET /exchange/bank-accounts'
description: "Lista todas as contas bancárias (PIX) do usuário autenticado"
---

## Visão Geral

Retorna todas as contas bancárias cadastradas para o usuário. É necessário o header `X-API-Key`.

## Parâmetros

<ParamField path="headers" type="object">
  <ParamField path="X-API-Key" type="string" required>
    Chave de API para autenticação
  </ParamField>
  <ParamField path="Content-Type" type="string">
    application/json
  </ParamField>
</ParamField>

<ParamField path="query" type="object">
  <ParamField path="page" type="string">
    Número da página (default: "1")
  </ParamField>
  <ParamField path="pageSize" type="string">
    Itens por página (default: "10")
  </ParamField>
  <ParamField path="name" type="string">
    Filtro parcial por nome (case-insensitive)
  </ParamField>
  <ParamField path="pix_key" type="string">
    Filtro parcial por chave PIX (case-insensitive)
  </ParamField>
  <ParamField path="type" type="string">
    Filtro exato por tipo de conta (ex: "pix")
  </ParamField>
</ParamField>

## Resposta

<ResponseField name="success" type="boolean">
  Indica se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem descritiva
</ResponseField>

<ResponseField name="data" type="array">
  Lista de contas bancárias
</ResponseField>

<ResponseField name="pagination" type="object">
  <ResponseField name="total" type="number">
    Total de registros
  </ResponseField>
  <ResponseField name="page" type="number">
    Página atual
  </ResponseField>
  <ResponseField name="pageSize" type="number">
    Itens por página
  </ResponseField>
  <ResponseField name="totalPages" type="number">
    Total de páginas
  </ResponseField>
  <ResponseField name="hasNextPage" type="boolean">
    Indica se há próxima página
  </ResponseField>
  <ResponseField name="hasPreviousPage" type="boolean">
    Indica se há página anterior
  </ResponseField>
</ResponseField>

## Exemplos

**cURL**

### Exemplos de uso

- Sem filtros (página 1, 10 itens):  
  `GET /exchange/bank-accounts?page=1&pageSize=10`

- Com filtro por nome e tipo (exemplo solicitado):  
  `GET /exchange/bank-accounts?page=2&pageSize=20&name=Conta&type=pix`

- Filtro por pix_key:  
  `GET /exchange/bank-accounts?pix_key=12345678900`

**cURL (com paginação)**

```bash
curl --request GET \
  --url "https://api.seudominio.com/exchange/bank-accounts?page=1&pageSize=10" \
  --header 'X-API-Key: sua-api-key-aqui'
```
