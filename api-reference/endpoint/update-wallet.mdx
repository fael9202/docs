---
title: "Atualizar Carteira"
openapi: 'PUT /exchange/wallets/{address}'
description: "Atualiza os dados de uma carteira cripto existente"
---

## Visão Geral

Atualiza os dados de uma carteira cripto existente. Atualmente, apenas o nome da carteira pode ser atualizado.

## Parâmetros

<ParamField path="path" type="object">
  <ParamField path="address" type="string" required>
    Endereço da carteira cripto
  </ParamField>
</ParamField>

<ParamField path="query" type="object">
  <ParamField path="network" type="string" required>
    Rede da criptomoeda (ex: "polygon", "polygon_amoy", "ethereum", "tron")
  </ParamField>
</ParamField>

<ParamField path="body" type="object">
  <Expandable title="UpdateWalletDto">

    <ParamField path="name" type="string">
      Novo nome para a carteira
    </ParamField>

    <ParamField path="address" type="string">
      Endereço da Carteira
    </ParamField>

  </Expandable>
</ParamField>
