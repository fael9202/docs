---
title: "Buscar Conta Bancária por ID"
openapi: 'GET /exchange/bank-accounts/{id}'
description: "Recupera uma conta bancária específica pelo ID"
---

## Visão Geral

Retorna os dados de uma conta bancária PIX pelo ID. Requer header `X-API-Key`.

## Parâmetros

<ParamField path="headers" type="object">
  <ParamField path="X-API-Key" type="string" required>
    Chave de API para autenticação
  </ParamField>
</ParamField>

<ParamField path="path.id" type="string" required>
  ID da conta bancária
</ParamField>

## Resposta

<ResponseField name="success" type="boolean">
  Indica se a operação foi bem-sucedida
</ResponseField>

<ResponseField name="message" type="string">
  Mensagem descritiva
</ResponseField>

<ResponseField name="data" type="object">
  Dados da conta bancária
</ResponseField>

## Exemplos

**cURL**

```bash
curl --request GET \
  --url https://api.seudominio.com/exchange/bank-accounts/1 \
  --header 'X-API-Key: sua-api-key-aqui'
```

**Resposta (200 OK)**

```json
{
  "success": true,
  "message": "Bank account retrieved successfully",
  "data": {
    "id": 1,
    "userId": 123,
    "type": "pix",
    "name": "Conta Principal PIX",
    "pix_key": "12345678900",
    "created_at": "2025-12-23T10:00:00.000Z",
    "updated_at": "2025-12-23T10:00:00.000Z"
  }
}
```

## Erros Possíveis

- 404 NOT_FOUND — Conta não encontrada
- 400 BAD_REQUEST — Conta não pertence ao usuário
