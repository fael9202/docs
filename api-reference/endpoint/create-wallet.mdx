title: "Criar Carteira"
openapi: 'POST /exchange/wallets'

---

## Visão Geral

Cria uma nova carteira cripto associada à sua conta. A carteira será usada para receber automaticamente as criptomoedas após a confirmação de pagamentos PIX.

<Note>
  Cada carteira é única e está associada a uma rede específica (Bitcoin, Ethereum, etc.).
</Note>

## Parâmetros

<ParamField path="body" type="object">
  <Expandable title="CreateWalletDto">

    <ParamField path="network" type="string" required>
      Rede da criptomoeda (ex: "polygon", "polygon_amoy", "ethereum", "tron")
    </ParamField>

    <ParamField path="address" type="string" required>
      Endereço da carteira cripto
    </ParamField>

    <ParamField path="name" type="string" required>
      Nome da carteira (ex: "Minha Carteira Bitcoin")
    </ParamField>

  </Expandable>
</ParamField>
