## Bem-vindo

A API da 3x Change permite que você integre facilmente a conversão de PIX para criptomoedas em sua aplicação. Com ela você pode criar carteiras cripto, consultar cotações, gerar pagamentos PIX e receber automaticamente as criptomoedas.

## Autenticação

Para autenticar as requisições à API, você precisa incluir duas chaves nos headers das requisições:

- `api_key`
- `api_secret`

Essas credenciais são únicas para cada conta e podem ser encontradas no painel administrativo da 3x Change.

<Card
  title="Acessando Minhas Credenciais"

<Warning>

</Warning>
